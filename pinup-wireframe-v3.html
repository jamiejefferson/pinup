üéØ Interaction Summary

Panel Toggle

  • Default: Closed (clean viewing)
  • Click "Comments (n)" to open
  • When open: dots appear on elements
  • When closed: dots hidden
  • Mobile: panel is full-screen overlay

Version Switching

  • Dropdown in top bar
  • Loads new prototype URL
  • Comments filter to that version
  • Dots only show for current version

Adding Comments

  • Click anywhere on prototype
  • Modal shows element selector
  • Type feedback, click "Pin"
  • Dot appears (if panel open)

Deleting Comments

  • Click trash icon on comment
  • Confirmation dialog appears
  • Shows comment preview
  • Confirm to delete permanently
